# Security Policy

## Supported Versions

This repository is maintained on a best-effort basis. When security issues
are identified, they will be addressed in the latest version.

## Reporting a Vulnerability

If you discover a security vulnerability:

- Do **not** open a public issue.
- Privately contact the maintainers with:
  - A detailed description of the issue
  - Steps to reproduce
  - Any potential impact you have identified

Responsible disclosure is appreciated and helps keep the Autokirk ecosystem safe.
